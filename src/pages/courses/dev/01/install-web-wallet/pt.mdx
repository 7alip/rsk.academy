Para obter criptomoedas, precisamos de um endereço / carteira para armazená-los.

Existem diversas formas de se gerar um endereço,
faremos isto a partir de um mnemônico em uma carteira.

Aqui estão algumas web wallets:

- [Metamask](https://metamask.io/)
- [Nifty](https://www.poa.network/for-users/nifty-wallet)

Estas carteiras geram o mnemônico para você.

Em nosso curso utilizaremos Nifty wallet.

## Nifty Wallet

A carteira Nifty é uma carteira da web que facilita fazer transações com suas contas.

Também pode ser usada com as redes RSK porque já está configurada para isso.
É uma extensão do Chrome.

[Nifty](https://www.poa.network/for-users/nifty-wallet) wallet
foi desenvolvida pela rede POA, que é uma sidechain do Ethereum.
Em outro momento do curso falaremos sobre sidechains.

## Crie uma carteira usando Nifty Wallet

Vá em

[Nifty wallet](https://www.poa.network/for-users/nifty-wallet)

clique no link **Chrome extension**.

![Nifty wallet install](/img/courses/dev/01/install-web-wallet/image-10.png)

Clique em **Add to Chrome**.

![Nifty wallet add to Chrome](/img/courses/dev/01/install-web-wallet/image-11.png)

Veja a mensagem de instalação bem-sucedida no canto superior esquerdo:

![Nifty wallet installed](/img/courses/dev/01/install-web-wallet/image-12.png)

Aceite os termos de uso

![Nifty wallet use terms](/img/courses/dev/01/install-web-wallet/image-13.png)

Defina uma senha

![Nifty wallet password](/img/courses/dev/01/install-web-wallet/image-14.png)

## Mnemônico

Copie seu mnemônico.

> Esta é a parte mais importante!
> Se alguém souber seu mnemônico, pode roubar sua carteira.
> Você pode esquecer a senha, mas o mnemônico nunca.
> Sem ele, você não tem mais acesso à sua carteira.

![Mnemonic](/img/courses/dev/01/install-web-wallet/image-15.png)

Por exemplo, meu mnemônico é:

```
shoot gate legend member sniff asthma smart slot famous call marriage math
```

Espero que você não roube minhas criptos, afinal, estou te ensinando :)

No canto superior direito, escolha a rede **RSK Testnet**.

![Choose network](/img/courses/dev/01/install-web-wallet/image-16.png)

Veja a rede selecionada, é a RSK Testnet.

![Network selected](/img/courses/dev/01/install-web-wallet/image-17.png)

## Conclusão

Pronto!
Você já criou sua primeira carteira.

Na próxima lição, você conectará sua carteira aqui na RSK Academy :)
