## RSK faucet tR-BTCs

El Smart Bitcoin (R-BTC) es el token utilizado para pagar la ejecución de transacciones en RSK.
En el faucet puede obtenerlo para RSK Testnet.

## Obter testnet R-BTCs en el faucet

Puedes solicitar Testnet R-BTC en
[faucet.testnet.rsk.co](https://faucet.testnet.rsk.co/).

Copia la dirección de tu billetera

![Copy address from Metamask](/images/image-03.png)

Ingresa el address de tu billetera y completa el CAPTCHA.

Por ejemplo, voy a obtener algunos tR-BTC para esta dirección:
`0xCD70794c2f3c657310EF13b6Ff3EC2d112513b39`.

![Captcha](/images/image-12.png)

Espera unos segundos ...

![Wait a few seconds](../../images/wallet-rsk-faucet/image-04.png)

![Received some R-BTCs](../../images/wallet-rsk-faucet/image-13.png)

Puedes ver el hash de la transacción, como este ejemplo:
[`0xe7a25985f019482d362a3be908f1c0b3dee612fcc78716b6a341d8ad6138ea95`](https://explorer.testnet.rsk.co/tx/0xe7a25985f019482d362a3be908f1c0b3dee612fcc78716b6a341d8ad6138ea95)

¡Ahora tengo 0.05 tRBTC!
