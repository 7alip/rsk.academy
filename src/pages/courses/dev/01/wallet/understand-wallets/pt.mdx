TODO

Para ter cryptos, precisamos de um endereço / wallet para armazená-los.

A melhor forma de criar uma wallet é a partir de um mnemônico,
utilizando o padrão definido em [BIP39](https://github.com/bitcoin/bips/blob/master/bip-0039.mediawiki)

Existem algumas formas de fazer isto.

Uma delas é criar com uma wallet web, como [Metamask](https://metamask.io/) ou [Nifty](https://www.poa.network/for-users/nifty-wallet).
Estas carteiras geram o mnemônico para você.
