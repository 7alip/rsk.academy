## Testnet

Una testnet (red de prueba) es una red con las mismas características que la red principal de un Blockchain,
pero sus criptomonedas y tokens no valen dinero real.

## RSK faucet tR-BTCs

El Smart Bitcoin (R-BTC) es el token utilizado para pagar la ejecución de transacciones en RSK.
En el faucet puede obtenerlo para RSK Testnet.

## Obter testnet R-BTCs en el faucet

Puedes solicitar Testnet R-BTC en
[faucet.testnet.rsk.co](https://faucet.testnet.rsk.co/).

Copia la dirección de tu billetera

![Copy address from Nifty](/img/courses/dev/01/rsk-faucet/image-01.png)

Ingresa el address de tu billetera y completa el CAPTCHA.

Por ejemplo, voy a obtener algunos tR-BTC para esta dirección:

**0xa63375173fFA44725b5CDe4dCfdE7D6Ccf5FFFaa**

![Captcha](/img/courses/dev/01/rsk-faucet/image-02.png)

Espera unos segundos ...

![Wait a few seconds](/img/courses/dev/01/rsk-faucet/image-03.png)

¡Y recibes tR-BTCs!

![Received some R-BTCs](/img/courses/dev/01/rsk-faucet/image-04.png)

Puedes ver el hash de la transacciónen el **explorer**, como este ejemplo:

[0x37e7ccc7fa73a748aa073ca5097ee9c0ffdbda9aafc4f61e6a71587328c479d2](https://explorer.testnet.rsk.co/tx/0x37e7ccc7fa73a748aa073ca5097ee9c0ffdbda9aafc4f61e6a71587328c479d2)

![RSK explorer](/img/courses/dev/01/rsk-faucet/image-05.png)

¡Ahora tengo 0.05 tRBTC!

![tR-BTCs at Nifty wallet](/img/courses/dev/01/rsk-faucet/image-06.png)
