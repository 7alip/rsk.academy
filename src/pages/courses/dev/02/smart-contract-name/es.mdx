# Crea un smart contract

1. Haz clic en el segundo botón en el lado izquierdo - file explorer 
2. Haz clic en `create a new file`

![create a new file](/img/courses/dev/02/smart-contract-name/image-01.png)

Nombre del archivo / file name: `Name.sol`

![filename Name.sol](/img/courses/dev/02/smart-contract-name/image-02.png)

Copia este ejemplo:

```solidity
// SPDX-License-Identifier: MIT
pragma solidity 0.8.1;

contract Name {
    string private name;
    address public owner;

    constructor() {
        owner = msg.sender;
        name = "Your fullname";
    }

    function getName() public view returns (string memory) {
        return name;
    }
}
```

Y pégalo aquí:

![Name.sol](/img/courses/dev/02/smart-contract-name/image-03.png)

## Name.sol

Este contrato inteligente tiene:

* Una variable privada `name` para almacenar tu nombre
* Una variable publica `owner` para almacenar la dirección del creador del contrato inteligente
* `constructor`, que se ejecuta solo en el momento de publicar el smart contract
* Una función `getName()` para devolver el nombre almacenado en la variable `name`

## Actualiza tu nombre

Dentro del `constructor`, en la línea 9, reemplace el mensaje "Your fullname" con su nombre completo.
Por ejemplo, cambiaré a "Solange Gueiros":

```solidity
name = "Solange Gueiros";
```

Este será el resultado:

![name Solange Gueiros](/img/courses/dev/02/smart-contract-name/image-04.png)

![compilation successful](/img/courses/dev/02/smart-contract-name/image-06.png)

Si no lo ha habilitado:

2. Clique no botão Compile Name.sol
3. Verifica el letrero verde en el tercer botón con el mensaje: compilation successful

## Publicar en JavascriptVM

En el panel lateral izquierdo, ve al botón `Deploy and run transactions`.

![Deploy and run transactions](/img/courses/dev/02/smart-contract-name/image-08.png)

En este momento solo tenemos un contrato inteligente, 
por lo que se selecciona automáticamente en el dropbox.
