## Remix

Remix é uma ferramenta online desenvolvida para o Blockchain Ethereum. É um IDE (Integrated Development Environment - ambiente de desenvolvimento integrado) usado para escrever, compilar, publicar e depurar código fonte em Solidity.
Dado que a RVM - RSK Virtual Machine é compatível com o Ethereum EVM, o Remix pode ser conectado diretamente ao nó local RSK e, com essa conexão, utilizado para publicar contratos inteligentes no Blockchain RSK.

Vá em [http://remix.ethereum.org/](http://remix.ethereum.org/)

![remix.ethereum.org](/img/courses/dev/02/remix/image-08.png)

Na página inicial - home / welcome page, escolha o ambiente / environment `Solidity`.

![Remix environment Solidity](/img/courses/dev/02/remix/image-09.png)

### Terminal

No Remix, na parte abaixo, à direita, existe um terminal com algumas bibliotecas disponíveis.

Você pode enviar comandos e transações por aqui. Também apresenta o resultado das transações e / ou chamadas às funções de smart contracts.

> Esta área de retorno é muito importante para acompanhar os resultados!

![remix terminal](/img/courses/dev/02/remix/image-10.png)

### Solidity compiler

Clique no 3o botão do lado esquerdo - Solidity compiler

![remix solidity compiler](/img/courses/dev/02/remix/image-11.png)

Habilite o `auto-compile` para facilitar sua vida, compilando automaticamente os smart contracts durante a edição no Remix.

![auto-compile](/img/courses/dev/02/remix/image-12.png)

## Conectando o Remix a RSK Testnet através de uma carteira Nifty ou Metamask

Com a RSK network selecionada em Nifty ou Metamask...

No painel à esquerda, clique no botão `Deploy and run transactions`.
Atualmente é o 4o botão.

![Deploy and run transactions](/img/courses/dev/02/remix/image-13.png)

Em Environment, escolha `Injected Web3`

![Injected Web3](/img/courses/dev/02/remix/image-14.png)

Injected Web3 conecta Remix com a conta ativa / selecionada na carteira.

### No Metamask

> Translate

Se abrirá una ventana popup de Metamask para autorizar la conexión.

Primero seleccione la cuenta y haz clic en botón Next. Probablemente solo tengas una.

![Metamask select accounts](/img/courses/dev/02/remix/image-15.png)

Después haz clic en botón Connect.

![Metamask connect](/img/courses/dev/02/remix/image-16.png)

### Remix conectado

En Remix, en Deploy and run transactions, mira el Custom Network

![Injected Web3 - ChainID 31](/img/courses/dev/02/remix/image-17.png)

ChainID 31 foi definida na configuração da RSK Testnet no Metamask.

Tudo pronto! Agora é possível fazer a publicação de qualquer smart contract na testnet RSK!
