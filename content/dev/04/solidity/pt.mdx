## Linguagem de programação Solidity

Solidity é uma linguagem de programação orientada a objetos, de alto nível,
usada para publicação de smart contracts.
Os contractos inteligentes governam o comportamento das contas e suas mudanças de estado no Ethereum.

Solidity é uma linguagem
[curly-bracket](https://en.wikipedia.org/wiki/List_of_programming_languages_by_type#Curly-bracket_languages),
isto é, os caracteres **{** e **}** são os delimitadores dos blocos de código da linguagem.

Sua sintaxe tem influência de C++, Python e JavaScript e
foi criada para ser executada na EVM - Ethereum Virtual Machine (Máquina Virtual Ethereum).

Solidity é uma linguagem _fortemente tipada_, suporta herança, bibliotecas e
tipos complexos definidos pelo usuário, entre outros recursos.

Esta definição de Solidity foi traduzida de [Solidity docs](https://docs.soliditylang.org/).
