## Remix

Remix é uma ferramenta online desenvolvida para o Blockchain Ethereum.
É um IDE (Integrated Development Environment - ambiente de desenvolvimento integrado) usado para escrever, compilar, publicar e depurar código fonte em Solidity.
Pode ser conectado a uma web Wallet e, com essa conexão, utilizado para publicar contratos inteligentes em diversas redes.

Vá em
[http://remix.ethereum.org/](http://remix.ethereum.org/)

![remix.ethereum.org](/img/courses/dev/02/remix/image-01.png)

Na página inicial - home / welcome page, escolha o ambiente: **environment Solidity**.

![Remix environment Solidity](/img/courses/dev/02/remix/image-02.png)

## Terminal

No Remix, na parte abaixo, à direita, existe um terminal com algumas bibliotecas disponíveis.

Você pode enviar comandos e transações por aqui.
Também apresenta o resultado das transações e / ou chamadas às funções de smart contracts.

> Esta área de retorno é muito importante para acompanhar os resultados!

![remix terminal](/img/courses/dev/02/remix/image-03.png)

## Solidity compiler

Clique no 3o botão do lado esquerdo: **Solidity compiler**.

![remix solidity compiler](/img/courses/dev/02/remix/image-04.png)

Habilite o **auto-compile** para facilitar sua vida, compilando automaticamente os smart contracts durante a edição no Remix.

![auto-compile](/img/courses/dev/02/remix/image-05.png)

## JavascriptVM

No painel à esquerda, clique no botão **Deploy and run transactions**. Atualmente é o 4o botão.

![deploy and run transactions](/img/courses/dev/02/remix/image-10.png)

O Remix possui o ambiente **JavaScriptVM**, um simulador de Blockchain que acontece na memória do browser.

![JavaScriptVM](/img/courses/dev/02/remix/image-11.png)

### Accounts / contas

Este simulador possui diversos endereços / contas com Ethers fictícios que podemos escolher para publicar um smart contract ou interagir com ele.

Veja um exemplo da lista de contas:

![account](/img/courses/dev/02/remix/image-12.png)

Em algumas versões do Remix, a lista de contas era alterada a cada vez que você inicia o Remix ou faz uma atualização na página.

## Web3 Providers

### Web3 Injected

Web3 Injected é um provedor web3 injetado no browser por um plugin, como
[MetaMask](https://metamask.io/) ou [Nifty](https://www.poa.network/for-users/nifty-wallet).

## Conectando o Remix a RSK Testnet através de uma carteira Nifty ou Metamask

Com a RSK testnet já selecionada em sua web wallet, Nifty ou Metamask...

No Remix, no painel à esquerda, clique no botão **Deploy and run transactions**.

![Deploy and run transactions](/img/courses/dev/02/remix/image-13.png)

Em Environment, escolha **Injected Web3**.

![Injected Web3](/img/courses/dev/02/remix/image-14.png)

Injected Web3 conecta Remix com a conta ativa / selecionada na carteira.

### No Metamask

Caso esteja utilizando a web wallet Metamask, abrirá uma janela popup para autorizar a conexão.

Primero selecione a conta e clique no botão Next.
Provavelmente você terá apenas uma conta.

![Metamask select accounts](/img/courses/dev/02/remix/image-15.png)

Em seguida clique no botão Connect.

![Metamask connect](/img/courses/dev/02/remix/image-16.png)

### Remix conectado

No Remix, em **Deploy and run transactions**, veja **Custom Network**.

![Injected Web3 - ChainID 31](/img/courses/dev/02/remix/image-17.png)

**ChainID 31** foi definida na configuração da RSK Testnet da web wallet injected.
